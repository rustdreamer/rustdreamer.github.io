mac 无法连接 ssh 解决方案

Mar 5, 2018

报错：

Unable to negotiate with 192.168.1.101 port 29418: no matching key exchange method found. Their offer: diffie-hellman-group1-sha1

解决：修改~/.ssh/config，加入

Host *
KexAlgorithms +diffie-hellman-group1-sha1
